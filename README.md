# glofx-firmware-by-lsgd
For the updater software from GLO.
【中文说明】

使用方法:
  1. 用 Type-C 数据线连接效果器与电脑
  2. 打开 GLO FX Updater
  3. 等待状态栏显示 "Connected" (绿灯)
  4. 点击 "Check Update" 检查最新固件
  5. 如有新版本，点击 "Upgrade" 开始升级
  6. 等待进度条完成，效果器自动重启

注意事项:
  · 升级过程中请勿断开连接
  · 需要安装 CH340 驱动 (Windows)
    下载: https://www.wch.cn/downloads/CH341SER_EXE.html
  · macOS / Linux 通常无需额外安装驱动

如遇问题:
  · 检查 Type-C 线是否为数据线(非纯充电线)
  · 尝试更换 USB 接口
  · 或官网/公众号报检修

---

[English]

How to use:
  1. Connect your pedal to computer via Type-C cable
  2. Open GLO FX Updater
  3. Wait for "Connected" status (green light)
  4. Click "Check Update" to fetch latest firmware
  5. If a new version is available, click "Upgrade"
  6. Wait for progress bar to complete; pedal restarts

Notes:
  · Do not disconnect during upgrade
  · Windows users may need CH340 driver:
    https://www.wch.cn/downloads/CH341SER_EXE.html
  · macOS / Linux usually work without extra drivers

Troubleshooting:
  · Make sure your Type-C cable supports data (not charge-only)
  · Try a different USB port
  · Manual BOOTLOADER entry:
    Hold BOOTSEL -> Press RESET -> Release BOOTSEL
